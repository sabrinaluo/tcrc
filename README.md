# TCRC
### Notes
- no backend interaction
- no local storage, DON'T refresh
- available view: login, student
- user accounts are hardcoded
- available tutors are hardcoded

### Build Setup

``` bash
# install dependencies
npm install

# build for production with minification
npm run build

# start server
npm run start
```
